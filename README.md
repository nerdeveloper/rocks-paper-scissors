# GETAWAY TYPESCRIPT APP

This app was developed in Typescript as Coding challenge for GETAWAY.

## Usage

```bash
git clone https://github.com/nerdeveloper/rocks-paper-scissors.git
cd  rocks-paper-scissors
npm install
npm run serve
```

## Test

```bash
npm run test
```
